# angualr-study

项目依赖  nodejs npm  bower grunt
初次运行  下载依赖包
在项目根目录 npm install  然后 npm  update
然后 bower install  bower update

运行 gunt serve

可以看到大概的浏览器界面



项目目录介绍
app目录下为主要代码目录 其他为依赖 配置等等
index.html为项目入口

bower_compents 为项目框架，插件等等
其他均为资源目录

view放html文件  详情参考test.html
scripts app.js 内部为路由 模块加载等等
controller/文件为模块js
